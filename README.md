# JSER Open Source
JSER 是一个简洁自由的开源社区软件，采用宽松的 MIT 协议开源。

# 工具与环境
1. 运行环境 Node.js
2. 数据库 MongoDB
3. 编辑器 VSCode

# 获取与贡献
1. 注册 GitHub 
2. 安装 Node.js 
3. 安装 MongoDB
4. 从 GitHub clone 代码
5. 在 “终端” 或 “CMD” 中执行 npm install，安装依赖模块
6. 利用 VSCode 编辑代码，（当然也可以用任何喜欢的文本编辑工具）
7. 成为 [jser-dev](https://github.com/jser-dev) 成员，可以直接提交代码
8. 当然，任何人都可以通过 Pull Request 向 JSER 贡献代码。
